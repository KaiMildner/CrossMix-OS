busybox-1.36.1-patched_by_Cizia.7z
==================================
Based on the busybox 1.36.1 original source code modified by Cizia with previous patches from Tina version. More complete than the stock one, the .config file includes the pkill command required for PortMaster, as well as all other common Buzybox commands.


Note
====
For the TG5050 wifi drivers loading, do not enable "CONFIG_MODPROBE_SMALL" otherwise the target module path will not be found.
